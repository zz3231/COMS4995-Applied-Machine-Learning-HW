HW3 — Run on Google Colab (GPU), then sync back to your machine / GitHub
======================================================================

1. Upload HW3.ipynb to Colab (or clone your repo and open HW3/HW3.ipynb).

2. Runtime → Change runtime type → GPU (T4 is enough).

3. Run all cells (Runtime → Run all). First cell installs `tokenizers`; training uses CUDA automatically.

4. After completion, download every fig_*.png (Files panel on the left, or run the final "download" cell).

5. Optional: download the executed notebook (File → Download → .ipynb) and replace HW3.ipynb locally if you want outputs saved.

6. Place the PNGs next to report.tex, update any numeric results in report.tex if they differ, then:
     pdflatex report.tex

7. Push to public GitHub: HW3.ipynb, fig_*.png, report.tex, report.pdf, requirements.txt (per assignment).

Local CPU alternative:
     pip install -r requirements.txt
     python run_hw3_train.py
   (Set HW3_EPOCHS / HW3_ABLATE / HW3_SHORT_EP env vars to shorten runs.)
